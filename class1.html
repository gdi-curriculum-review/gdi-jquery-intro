Variables

Variables hold content

  • Think of boxes and the content of the box
  • Box label is the name of the variable
  • Box content is the value of the variable
  • Three main types: Words, numbers, true/false

    • Declare a variable (Give it a name) --> box label

    
                var bananas;

    Initialize variable (Give it a value) --> put something in the box

    
                bananas = 5;

    If/Else Statement

    You can use else to perform an alternative action if the "if" fails

    
              var budget = 5;
              if (budget > 3){
              console.log('Buy a banana!');
              } else {
              console.log('Who needs Bananas');
              }

    If/Else Statement

    You can use else if to have multiple choices

    
              var age = 20;
              if (age >= 35) {
              console.log('You can vote AND hold any place in government!');
              } else if (age >= 25) {
              console.log('You can vote AND run for the Senate!');
              } else if (age >= 18) {
              console.log('You can vote!');
              } else {
              console.log('You have no voice in government (yet)!');
              }

    Functions

    • A Functions is a re-usable collection of statements
    • Think of them as recipes. A recipe is a reusable list of instructions
    Declare a function 
    
 //sayHi is the label and the content is the list of instructions that we want to store and re-run
             function sayHi(){
             console.log('Hi!');
             }
    Call the function: run the instructions 
    
             sayHi();

    Arguments

    Functions can take arguments

    Arguments are variables we pass to the function

    The function takes these variables and perform some instructions using those variables

    

//the function sayHi has one argument (variable) called name we pass to it from outside the function
             function sayHi(name){
             console.log('Hi' + name + '!');
             }
             //the function takes variable "name" and displays: Hi Pip!
             var name = 'Pip';
             sayHi(name);
             //we can pass a String to the function directly
             sayHi('Mitch');

    Arguments

    Functions can take MULTIPLE named arguments

    
             function addNumbers(num1, num2){
             var result = num1 + num2;
             console.log(result);
             }

             addNumbers(5, 6);

             var number1 = 12;
             var number2 = 15;
             addNumbers(number1, number2);

    Return values

    Functions can return a value

    
             function addNumbers(num1, num2){
             var result = num1 + num2;
             return result; //Anything after this line won't be read
             }
             var sum  = addNumbers(5, 6);

    Variable Scope

    JavaScript variables have "function scope." They are visible in the function where they are defined

    A variable with "global" scope: 
    
             var result;
             function addNumbers(num1, num2){
             result = num1 + num2;
             return result; //Anything after this line won't be read
             }
             var sum  = addNumbers(5, 6);
             console.log(result); //will return 11 because the variable was defined outside the function
